# Go-ninja

## API Documentation

The application exposes interactive API documentation powered by Huma and Swagger UI.

Start the application:

```powershell
go run -race ./src
```

Open Swagger UI in a browser:

```text
http://localhost:8080/docs
```

The generated OpenAPI specification is also available at:

```text
http://localhost:8080/openapi.json
http://localhost:8080/openapi.yaml
```
